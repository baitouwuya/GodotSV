# 简单搜索功能测试

这是一个完全独立的 GDSV 搜索功能测试脚本，不依赖任何外部类或文件。

## 文件说明

- `simple_search_test.gd` - 主测试脚本
- `simple_search_test.tscn` - 测试场景文件

## 运行方式

### 方法 1: 在 Godot 编辑器中运行

1. 在 Godot 编辑器中打开 `demo/` 项目
2. 双击打开 `tests/search/simple_search_test.tscn` 场景
3. 按 F5 或点击"播放场景"按钮

### 方法 2: 作为主场景运行

1. 在 **项目设置 -> 应用程序 -> 运行** 中设置主场景为 `tests/search/simple_search_test.tscn`
2. 按 F5 运行项目

## 测试覆盖

该脚本包含以下测试场景：

1. **基础文本搜索** - 查找名字 'Alice'
2. **大小写敏感搜索** - 区分/不区分大小写搜索
3. **正则表达式搜索** - 使用正则表达式匹配城市名、年龄等
4. **空结果测试** - 搜索不存在的项目、空字符串边界情况
5. **多结果测试** - 查找标签 'dev'、活跃用户等
6. **特定列搜索** - 指定列进行搜索

## 测试数据

测试数据内置在脚本中，无需外部文件。数据包含以下字段：

- id: 整数 ID
- name: 姓名
- age: 年龄
- city: 城市
- active: 是否活跃
- score: 分数
- tags: 标签

## 预期输出

- 每个测试的场景描述

- 测试结果（通过/失败）
- 汇总统计（总数、通过数、失败数、成功率）
